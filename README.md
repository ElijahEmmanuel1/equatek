# Equatek

Equatek is a study web app for CPGE MPSI and MP students, centered on maths, physics, and chemistry revision.

## Stack

- React 19
- TypeScript
- Vite

## Development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Build the production version:

```bash
npm run build
```

Lint the project:

```bash
npm run lint
```
